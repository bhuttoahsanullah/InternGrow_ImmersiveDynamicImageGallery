# 📸 Immersive Dynamic Image Gallery

A modern and responsive image gallery built using **HTML, CSS, and JavaScript**. The project features a responsive CSS Grid layout, interactive hover animations, a modal lightbox for viewing images, and dynamic filtering through a live search bar and category tabs—all without page reloads.

## Features

* Responsive CSS Grid layout
* Interactive image gallery
* Live search filter
* Category filtering (All, Nature, Architecture, Tech)
* Instant filtering without page reload
* Smooth CSS hover animations
* Modal lightbox image preview
* Mobile, tablet, and desktop responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## 📖 How to Run

1. Clone this repository:
```bash
git clone https://github.com/bhuttoahsanullah/InternGrow_ImmersiveDynamicImageGallery.git
```
2. Open the project folder.
3. Open `index.html` in your web browser.

No additional libraries or installation are required.

## Functionality

* Browse images in a responsive gallery.
* Click an image to open it in a modal lightbox.
* Search images instantly using the search bar.
* Filter images by category:

  * Nature
  * Architecture
  * Tech
* Combine category filters with search for faster browsing.
